Thuộc tính trong html
Thuộc tính trong html sẽ có tác dụng bổ sung thông tin cho thẻ html
ví dụ thẻ img có thuộc tính src="" đường dẫn của hình ảnh
ví dụ thẻ a có thuộc tính href="" chứa liên kết thẻ a
thẻ form có thuộc tính name="" , method="post" , action=""
thuộc tính class và id
thuộc tính class gom nhóm các thẻ html
thuộc tính id định danh các thẻ html là duy nhất
thuộc tính luôn xuất hiện trong thẻ mở của html
trong css mà muốn chọn 1 class trong html
thì sẽ dùng cú pháp là .class
trong css sau khi chọn được class mong muốn có cú pháp {}
trong {} chứa các thuộc tinh và giá trị của css
